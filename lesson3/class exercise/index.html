תרגיל כיתה:
1. צרו תיבת טקסט לקבלת שם מלא של משתמש
צרו לחצן שבלחיצה עליו מבצע:
א. הצגה בהודעה עבור השם המלא שנקלט בתיבה
ב. שמירה ב-LocalStorage של השם המלא
בטעינת הדף יש להחזיר בחזרה את השם המלא מהזיכרון המקומי לתיבת הטקסט

2. צרו את הפונקציה הבאה:
function paintPage(callback){
    alert("painting the page");
    callback();// the callback will paint the page in some color
}
