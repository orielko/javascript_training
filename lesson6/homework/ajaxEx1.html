שיעורי בית –הצגת משתמשים


צרו דף שבטעינה שלו מבצע גלישת AJAX לכתובת הבאה המכילה רשימת משתמשים: 
https://jsonplaceholder.typicode.com/users
והציגו את רשימת המשתמשים בטבלה.

•	שם
•	שם משתמש
•	אימייל
•	טלפון
•	עיר
•	רחוב
•	מיקוד
•	שם החברה

name user name email phone city street zip code company name
