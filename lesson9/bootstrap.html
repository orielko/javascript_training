In the past we used media queries in order to give a different layout for a responsive UI

        Media queries- used to do the CSS only if it satisfy the condition
        @media (width >100) and (width < 600){
            body{
                color:red;
            }
        }

        Bootstrap gives us 3 things:
        1. Bootstrap Grid system
        2. Bootstrap css
        3. Bootstrap components

        In bootstrap we only care for the width of the screen.

        5 Grid Tiers:
            1. Extra Small: >= 0px ,Initials: .col-
            2. Small: >= 576px ,Initials: .col-sm-
            3. Medium: >= 768px ,Initials: .col-md-
            4. Large: >= 992px ,Initials: .col-lg-
            5. Extra Large: >= 1200px ,Initials: .col-xl-
        "Mobile first design"- takes the smaller, until a bigger one is defined (doesn't work for the oposite direction),
        so we first design the mobile.

        To work with Bootstrap we need to create a container of a Grid, with rows, and inside the rows:columns
        
        -bootstrap classes: container,row,col
        -container-fluid: is for more fluid when we change the size
        -bootstrap always contains 12 columns, this is why col-6 will be half size (from 12) the rest will take all the remaining 6 columns.
         for more we need to make it nested
